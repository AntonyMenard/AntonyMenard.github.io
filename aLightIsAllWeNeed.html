A Light Is All We Need

Third person shooter Vampire Survivor-like in Unity

Gameplay

The game is not yet finished, but the end game is to have a Vampire Survival-like gameplay with a more active role for the player, who will be able to aim and use certain abilities themself. Visibility of the level layout is limitied, but is revealed as the game goes on.

Development

The game is my current personal project. The character's movement needs to be adjusted to feel better but is good enough to test other elements of the game. Basic enemies and energy orb can already spawn, and the character can attack, meaning that only the death of the character is needed to complete a basic gameplay loop. After the death of the player, the next element to do is the power upgrade system where the player will be able to learn more "spells" and upgrade those he already has when he collects enough energy orb.
